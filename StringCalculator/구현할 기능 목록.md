# 구현 기능

### 문자열로 부터 숫자 추출하기

- 구분자가 기본 구분자(`,`, `:`)인 경우 ✅

  Test) 정규 표현식을 통한 기본 구분자 문자열인지를 확인.

- 구분자가 커스텀 구분자인 경우

  Test) 정규 표현식을 통한 `// \n` 을 포함한 커스텀 구분자 문자열인지를 확인. ✅
  
  Test) 커스텀 구분자가 모두 동일한지 확인
  ex)`//<\n1:2:3` 은 예외 ✅
  
  

### 예외처리하기 

- 음수가 하나 이상 포함되어 있는 문자열에 대하여 `RuntimeException` 예외처리  ✅

- 유효하지 않은 문자열에 대한 예외처리하기 ✅

  >ex) "asdfasd", "1,1#1"
